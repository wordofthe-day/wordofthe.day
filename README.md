# wordofthe.day

A minimal website that delivers a new vocabulary word every day. Each word comes with its pronunciation, definition, an example sentence, and audio so you can hear how it's said.

## How it works

A new word is selected daily at **00:00 UTC** across all supported languages. The countdown on the site shows how long until the next word drops.

## Supported languages

- English

Want another language? [Open an issue](https://github.com/wordofthe-day/wordofthe.day/issues) and let me know.
